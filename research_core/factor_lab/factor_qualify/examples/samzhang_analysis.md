## 二、核心问题：为什么全都挂了

### 2.1 通用失败点：WFA全部不通过

**17个因子在S2阶段（WFA滚动窗口）全部失败。** 这是最致命的共同问题。

WFA要求两个条件同时满足：
- **majority pass**：>50%的滚动窗口Sharpe > 0
- **catastrophic veto**：没有任何单窗口Sharpe < -2 或 MDD > 50%

所有17个因子全部触发了catastrophic veto。原因：单因子信号的日频噪声太大。纯粹按一个因子排序做多空，在某些月份会遭遇极端亏损——不是因为因子逻辑错了，而是因为**单因子无法在日频层面稳定击败市场噪音**。

这跟因子本身的IC质量无关——ret_5d的ICIR高达3.5，信号方向极其显著。但日频多空的MDD依然炸裂。原因是：截面IC反映的是"排序和收益的平均相关性"，但多空组合在极端日子里会被少数股票的异常波动打穿。

### 2.2 A股日频的反转主导

17个因子中12个的IC方向为**负**（因子值越高，未来收益越低），即反转信号主导。只有illiquidity、min_ret_1m、reversal（刻意取反后）表现为正向。

这不是bug。A股日内/日频层面，散户追涨杀跌造成的短期反转效应压倒动量效应。所以ret_5d和reversal本质上是一个因子（互为相反数），ICIR都是3.5。

### 2.3 OOS衰减严重

OOS（2021-2025）期间几乎所有因子的SR都低于IS或转负。典型案例如：
- bb_position：IS -0.10 → OOS -0.92，衰减率 8.57（暴雷）
- volume_ratio：IS 0.51 → OOS -0.72，衰减率 2.43（失效）
- ret_5d：IS 0.17 → OOS -0.51，衰减率 3.90（反向）

衰减不是"慢慢变弱"，而是**断崖式反转**——IS期间的方向在OOS期间完全反了。这说明2021年后的市场微观结构发生了根本性变化，旧规律不再适用。

## 三、各因子详细分析

### 3.1 最强IC的因子：ret_5d / reversal

| 指标 | 值 |
|------|-----|
| IC&#124;均值&#124; | 0.0375（日频，最强） |
| ICIR年化 | 3.5 |
| 方向 | 反转（短期涨→次日跌） |
| S0 | ✓ 通过 |
| IS SR | 0.17 |
| OOS SR | -0.51 |
| WFA | ✗ catastrophic veto |

**原理**：A股T+1制度和散户追涨杀跌创造了显著的短期反转效应。过去5天涨最多的股票，次日平均会跌。这个规律极其稳定（ICIR=3.5在日频属于顶级的统计显著性）。

**为什么不能实盘**：OOS完全失效。2021年前做多低5日收益+做空高5日收益能赚小钱（SR=0.17），2021年后这个策略亏钱（SR=-0.51）。可能是因为量化基金大量套利后，这个最简单的反转策略已经被拥挤交易吃掉了。

### 3.2 唯一OOS为正的因子：illiquidity

| 指标 | 值 |
|------|-----|
| IC&#124;均值&#124; | 0.0203 |
| ICIR年化 | 2.5 |
| 方向 | 正向（流动性越差→未来收益越高，非流动性溢价） |
| S0 | ✓ 通过 |
| IS SR | 0.09 |
| OOS SR | 0.62 |
| WFA | ✗ catastrophic veto |

**原理**：Amihud (2002) 的非流动性溢价——流动性差的股票需要更高的预期收益来补偿交易成本。因子定义是 |日收益|/成交额，捕捉的是"每单位成交额带来的价格冲击"。

**为什么OOS反而变好**：2021-2025年市场风格从小票成长转向价值+红利，小票的流动性溢价被重新定价。IS期间这个溢价被成长股泡沫压制（流动性差的股票被抛弃），OOS期间风格回归，溢价释放。

**为什么不能实盘**：仍然是WFA catastrophic veto。日频层面的MDD太大——非流动性因子选出来的股票天然是低流动性小票，极端日子跌起来没有流动性承接，多空组合的做空端（做空高流动性大票）在大涨日会爆亏。

### 3.3 次强因子：vol_convergence / volume_ratio

| 指标 | vol_convergence | volume_ratio |
|------|:---:|:---:|
| ICIR年化 | 3.1 | 3.2 |
| IS SR | 0.53 | 0.51 |
| OOS SR | 0.06 | -0.72 |
| WFA | ✗ | ✗ |

**原理**：成交量因子——短期成交量相对长期成交量的比值。高比值意味着"最近放量"，A股里放量通常出现在短期顶部（散户蜂拥买入），所以这两个因子都是反转方向（高成交量→次日跌）。

**为什么vol_convergence表现更好**：它用的是5日/20日均量比，信号更平滑。volume_ratio是当日/5日均量，太敏感，OOS直接崩。

### 3.4 基本无用：bb_position / rsi_14 / amplitude_1m / high_low_1m

这4个因子的共同特点是：ICIR不低（1.3-2.7），但多空SR全是负数。原因是它们的信号方向和收益的关系在A股日频上不是线性的——超买/超卖信号在极端值有效，但中段是噪音。

单一阈值（如RSI>70做空）可能有效，但线性排序无效。这些因子需要**非线性处理**，不是直接排序能用的。

## 四、根本原因：单因子不可单独使用

17个因子全部失败，不是因为这17个因子本身是"坏因子"，而是因为**单因子验证的假设在日频A股不成立**。

单因子验证隐含假设：按因子排序做多空，收益应该稳定为正。但A股日频的噪音水平远高于这个假设能承受的范围。一个ICIR=3.5的因子（ret_5d），150个WFA窗口中仍有超过一半窗口亏钱——不是因为信号不存在，而是日频波动淹没了信号。

正确的用法是**多因子合成**——这正是samzhang8/multifactor_synthesis.py在做的事。17个因子各自提供了不同的截面信息，合成后噪音对消、信号增强。单个因子单独看是噪音，组合起来才是信号。

这个库的原始设计是对的：不单独使用任何一个因子，而是通过滚动IC加权把17个因子合成一个综合信号。
